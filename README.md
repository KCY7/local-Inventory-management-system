# Holychild Inventory System

A locally hosted inventory management application built with Django. This system provides a streamlined, centralized platform to track, update, and manage inventory data securely on a local network or machine.

## 🚀 Features
* **Local Hosting Environment:** Designed to run efficiently on a local machine without requiring external cloud databases.
* **Inventory Tracking:** Easily add, modify, and monitor stock levels and item details.
* **Lightweight Database:** Utilizes SQLite3 for reliable, out-of-the-box data storage.

## 🛠️ Tech Stack
* **Framework:** Django (Python)
* **Database:** SQLite3
* **Environment Management:** `venv`
